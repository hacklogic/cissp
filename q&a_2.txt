

题目1：James正在为他的组织制定一个灾难恢复计划，并希望确定在组织的系统发生故障后可接受的数据丢失量。James正在确定什么变量？
James is building a disaster recovery plan for his organization and would like to determine the amount of acceptable data loss after an outage. What variable is James determining?
A：SLA
SLA
B：RTO
RTO
C：MTD
MTD
D：RPO
RPO
答案:James正在确定的是 **D：RPO**。

**RPO（Recovery Point Objective，恢复点目标）** 是指在灾难发生后，组织可以接受的数据丢失量。它定义了在系统恢复后，数据可以恢复到的时间点。例如，如果RPO为1小时，意味着在灾难发生后，系统恢复时最多只能丢失1小时的数据。

其他选项的解释：
- **A：SLA（Service Level Agreement，服务级别协议）**：是服务提供商和客户之间的协议，规定了服务的质量和可用性标准。
- **B：RTO（Recovery Time Objective，恢复时间目标）**：是指在灾难发生后，系统或服务必须恢复的时间目标。
- **C：MTD（Maximum Tolerable Downtime，最大可容忍停机时间）**：是指组织可以容忍的系统或服务的最长停机时间。

因此，James正在确定的是 **RPO**。

题目3：Alex正在准备对其公司的网络系统进行渗透测试，为此他进行了招标。他想要最大程度地保证测试的有效性，并不关心真实的攻击。请问他在招标过程中需要进行什么类型的渗透测试？
Alex is preparing to solicit bids for a penetration test of his company’s network and systems. He wants to maximize the effectiveness of the testing rather than the realism of the test. What type of penetration test should he require in his bidding process?
A：黑盒
Black box
B：白盒
White box
C：灰盒
Gray box
D：零盒
Zero box
答案:在Alex的情况下，他想要最大程度地保证测试的有效性，而不关心真实的攻击。这意味着他希望测试人员能够尽可能全面地发现系统中的漏洞和弱点，而不需要模拟真实的攻击场景。因此，他应该选择**白盒测试（White box）**。

**白盒测试**是指测试人员在完全了解系统内部结构、代码和配置的情况下进行的测试。这种测试方式允许测试人员深入分析系统的各个方面，从而更有效地发现潜在的安全漏洞。由于测试人员拥有系统的详细信息，他们可以更全面地进行测试，而不需要花费大量时间进行信息收集和侦察。

其他选项的解释如下：
- **黑盒测试（Black box）**：测试人员对系统内部结构一无所知，模拟真实的攻击场景。这种方式更注重测试的真实性，但可能无法全面覆盖所有漏洞。
- **灰盒测试（Gray box）**：介于黑盒和白盒之间，测试人员拥有部分系统信息。这种方式在真实性和有效性之间取得平衡。
- **零盒测试（Zero box）**：这个选项并不是一个常见的测试类型，可能是干扰项。

因此，**B：白盒（White box）**是最适合Alex需求的渗透测试类型。

题目6：作为新的BYOD策略的一部分，Jim被要求能够识别员工自行带到公司的设备。这些设备并不会被加入到例如Active Directory这样的集中管理系统，下列哪项将会为Jim提供最佳方法来可靠地识别各个设备？
Jim has been asked to individually identify devices that users are bringing to work as part of a new BYOD policy. The devices will not be joined to a central management system like Active Directory, but he still needs to uniquely identify the systems. Which of the following options will provide Jim with the best means of reliably identifying each unique device?
A：记录各个系统的MAC地址
Record the MAC address of each system.
B：要求用户填写表格为每个系统进行注册
Require users to fill out a form to register each system.
C：利用端口扫描器对每个系统进行扫描
Scan each system using a port scanner.
D：通过Web注册系统采用设备指纹
Use device fingerprinting via a web-based registration system.
答案:在BYOD（Bring Your Own Device）策略中，识别员工自行带到公司的设备是一个重要的任务。由于这些设备不会被加入到集中管理系统（如Active Directory），因此需要一种可靠的方法来唯一地识别每个设备。以下是每个选项的分析：

A：记录各个系统的MAC地址  
MAC地址是网络接口的唯一标识符，通常不会改变。记录MAC地址是一种简单且可靠的方法来识别设备。然而，MAC地址可以被伪造或更改，因此这种方法并不是绝对可靠的。

B：要求用户填写表格为每个系统进行注册  
这种方法依赖于用户的诚实和准确性。用户可能会填写错误的信息，或者故意提供不准确的信息。因此，这种方法并不可靠。

C：利用端口扫描器对每个系统进行扫描  
端口扫描器可以识别设备上开放的端口和服务，但这并不能唯一地识别设备。此外，端口扫描可能会被视为侵入性行为，引起用户的不满或隐私问题。

D：通过Web注册系统采用设备指纹  
设备指纹是一种通过收集设备的多种信息（如浏览器类型、操作系统、屏幕分辨率、插件等）来唯一识别设备的技术。这种方法可以可靠地识别设备，并且不需要用户手动输入信息。设备指纹技术通常用于Web应用程序中，可以自动收集设备信息并生成唯一的标识符。

综上所述，**D：通过Web注册系统采用设备指纹** 是最佳的方法，因为它可以自动、可靠地识别每个设备，而不依赖于用户的手动输入或容易伪造的信息。

题目8：James希望确保其公司的备份能够在数据中心遭受灾难时幸免于难。以下哪个选项是解决此问题的最佳方法？
James wants to ensure that his company’s backups will survive a disaster that strikes the data center. Which of the following options is the best solution to this concern?
A：异地备份
Off-site backups
B：祖父/父/子备份分层系统
A grandfather/father/son backup tiering system
C：冗余备份系统
Redundant backup systems
D：SAN或NAS的快照
Snapshots to a SAN or NAS
答案:题目8的正确答案是：

**A：异地备份 (Off-site backups)**

### 解析：
James希望确保其公司的备份能够在数据中心遭受灾难时幸免于难。这意味着备份数据需要存储在远离主数据中心的地方，以防止灾难（如火灾、洪水、地震等）同时摧毁主数据中心和备份数据。

- **A：异地备份 (Off-site backups)**  
  这是最佳解决方案，因为异地备份将数据存储在远离主数据中心的位置，确保即使主数据中心遭受灾难，备份数据仍然安全可用。

- **B：祖父/父/子备份分层系统 (A grandfather/father/son backup tiering system)**  
  这是一种备份策略，用于管理不同时间点的备份数据，但它并不能解决数据中心灾难导致的数据丢失问题。

- **C：冗余备份系统 (Redundant backup systems)**  
  冗余备份系统可以提高数据的可用性和可靠性，但如果备份数据仍然存储在同一数据中心，仍然无法抵御灾难性事件。

- **D：SAN或NAS的快照 (Snapshots to a SAN or NAS)**  
  快照是一种快速备份技术，但如果快照数据存储在同一数据中心，仍然无法抵御灾难性事件。

因此，**异地备份 (Off-site backups)** 是最佳解决方案。

题目9：密码的安全性是Gabe所在组织的信息安全计划的重中之重，她很担心密码的安全性。以下哪一项可通过对用户进行身份验证来帮助Gabe最大限度地保证密码安全？
Gabe is concerned about the security of passwords used as a cornerstone of his organization’s information security program. Which one of the following controls would provide the greatest improvement in Gabe’s ability to authenticate users?
A：更复杂的密码 
More complex passwords
B：对用户进行安全教育以防止社会工程学攻击
User education against social engineering
C：多因素身份验证 
Multifactor authentication
D：多使用基于个人知识的安全问题
Addition of security questions based on personal knowledge
答案:题目要求选择一种能够最大程度提高Gabe对用户进行身份验证能力的控制措施。我们来逐一分析选项：

A：更复杂的密码  
虽然更复杂的密码可以增加密码的安全性，但它仍然依赖于单一的身份验证因素（即密码），因此无法显著提高身份验证的整体安全性。

B：对用户进行安全教育以防止社会工程学攻击  
用户教育可以帮助防止社会工程学攻击，但它并不直接提高身份验证的安全性，尤其是在密码泄露或被盗的情况下。

C：多因素身份验证  
多因素身份验证（MFA）要求用户提供两种或更多种不同类型的身份验证因素（例如密码和一次性验证码）。这种方法显著提高了身份验证的安全性，因为即使密码被泄露，攻击者仍然需要其他因素才能通过验证。

D：多使用基于个人知识的安全问题  
基于个人知识的安全问题通常作为密码恢复的辅助手段，但它们本身并不提供额外的安全性，尤其是在攻击者能够通过社会工程学或其他手段获取这些信息的情况下。

综上所述，**多因素身份验证**（选项C）能够最大程度地提高Gabe对用户进行身份验证的能力，因此是最佳选择。

**正确答案：C：多因素身份验证**

Packets with a source address from Angie’s public IP address block
B：目的地址为Angie公共IP地址的数据包
Packets with a destination address from Angie’s public IP address block
C：源地址不是Angie地址块的数据包
Packets with a source address outside Angie’s address block
D：源地址为Angie私有地址的数据包
Packets with a source address from Angie’s private address block
答案:在配置网络出口监控（egress monitoring）时，Angie的目的是确保只有合法的数据包能够离开她的网络进入互联网。为了增强安全性，她应该允许以下类型的数据包离开网络：

**D：源地址为Angie私有地址的数据包**
- 这些数据包的源地址是Angie的私有IP地址。私有IP地址不应该出现在互联网上，因为它们只在内部网络中使用。允许这些数据包离开网络可能会导致安全问题，因为它们可能被误认为是合法的出站流量。

因此，**A：源地址为Angie公共IP地址的数据包** 是应该被允许离开网络进入互联网的数据包类型。

题目16：以下哪种工具可提供不可否认性？
Which one of the following tools may be used to achieve the goal of nonrepudiation?
A：数字签名 
Digital signature
B：对称加密
Symmetric encryption
C：防火墙 
Firewall
D：IDS
IDS
答案:正确答案是 **A：数字签名**。

**解释：**
- **数字签名**（Digital signature）可以提供不可否认性（Nonrepudiation）。不可否认性是指发送方无法否认其发送过某条消息，接收方也无法否认其接收过某条消息。数字签名通过使用发送方的私钥对消息进行签名，接收方可以使用发送方的公钥验证签名的真实性，从而确保消息的来源和完整性。
  
- **对称加密**（Symmetric encryption）主要用于数据的保密性，但它无法提供不可否认性，因为加密和解密使用的是同一个密钥，无法区分发送方和接收方。

- **防火墙**（Firewall）主要用于网络安全，控制进出网络的流量，但它与不可否认性无关。

- **IDS**（Intrusion Detection System，入侵检测系统）用于检测网络中的异常行为或攻击，但它也无法提供不可否认性。

因此，**数字签名**是唯一能够提供不可否认性的工具。

答案:TOC/TOU（Time-of-Check to Time-of-Use）攻击是一种利用程序在检查资源状态和使用资源之间的时间差来进行攻击的技术。这种攻击通常发生在多线程或多进程环境中，攻击者通过改变资源的状态来利用这个时间差。

D：并发控制
Concurrency control

   - 有6个用户，每个用户需要与其他5个用户通信。
   - 每对用户之间需要一个独立的对称加密密钥。

   - 我们需要计算从6个用户中选取2个用户的组合数，因为每对用户需要一个密钥。

Sue’s organization recently failed a security assessment because their network was a single flat broadcast domain, and sniffing traffic was possible between different functional groups. What solution should she recommend to help prevent the issues that were identified?
A：使用VLAN 
Use VLANs.
B：更改所有系统的子网掩码
Change the subnet mask for all systems.
C：部署网关
Deploy gateways.
D：打开端口安全
Turn on port security.
答案:题目描述中提到，Sue的组织未能通过安全评估，原因是他们的网络是一个单一的平面广播域，并且不同的功能组之间可能会嗅探流量。这意味着网络中的所有设备都在同一个广播域中，缺乏隔离，导致安全风险增加。

### 选项分析：
- **A：使用VLAN**  
  VLAN（虚拟局域网）可以将网络划分为多个逻辑子网，从而隔离不同的功能组。通过使用VLAN，可以防止不同组之间的流量被嗅探，提高网络安全性。这是解决单一广播域问题的有效方法。

- **B：更改所有系统的子网掩码**  
  更改子网掩码可能会重新划分IP地址范围，但并不能解决广播域的问题。单一广播域仍然存在，不同组之间的流量仍然可能被嗅探。

  部署网关可以帮助不同网络之间的通信，但并不能直接解决单一广播域的问题。网关通常用于连接不同的网络，而不是隔离同一网络中的流量。

- **D：打开端口安全**  
  端口安全可以限制连接到特定端口的设备数量或MAC地址，但它并不能解决广播域的问题。端口安全主要用于防止未经授权的设备接入网络，而不是隔离流量。

### 正确答案：
**A：使用VLAN**  
通过使用VLAN，可以将网络划分为多个逻辑子网，隔离不同功能组之间的流量，从而防止流量嗅探，提高网络安全性。
