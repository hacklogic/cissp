题目2：Sally在传输模式下使用IPSec的ESP组件。她应该了解哪些关于传输模式的重要信息？
Sally is using IPsec’s ESP component in transport mode. What important information should she be aware of about transport mode?
A：传输模式提供整个IP数据包的完全加密
Transport mode provides full encryption of the entire IP packet.
B：传输模式添加一个新的未加密报头，以确保数据包到达其目的地
Transport mode adds a new, unencrypted header to ensure that packets reach their destination.
C：传输模式不加密数据包的头
Transport mode does not encrypt the header of the packet.
D：传输模式不提供加密，只有隧道模式提供加密
Transport mode provides no encryption; only tunnel mode provides encryption.
答案:在传输模式下使用IPSec的ESP（封装安全载荷）组件时，Sally应该了解以下重要信息：

C：传输模式不加密数据包的头
Transport mode does not encrypt the header of the packet.

解释：
- **传输模式**：在传输模式下，IPSec只加密IP数据包的**有效载荷**（即数据部分），而不加密IP头。这意味着IP头（包括源地址和目标地址）仍然是明文传输的。传输模式通常用于端到端的通信，例如在两个主机之间直接通信时使用。
  
- **隧道模式**：相比之下，隧道模式会加密整个IP数据包（包括IP头），并添加一个新的IP头。隧道模式通常用于在两个网络之间建立安全通道，例如在两个VPN网关之间通信时使用。

其他选项的解释：
- **A**：传输模式不提供整个IP数据包的完全加密，它只加密有效载荷部分。
- **B**：传输模式不会添加新的未加密报头，它只是保留原始的IP头。
- **D**：传输模式确实提供加密，但它只加密有效载荷部分，而不是整个数据包。

因此，正确答案是 **C**。

**A：RTO < MTD**

- 审计员（Auditor）：负责审查和评估数据保护措施的有效性，但不承担最终责任。

因此，正确答案是A：数据所有者（Data Owner）。

题目109：以下哪项控制在检测零日攻击方面最有效？
Which one of the following controls would be most effective in detecting zero-day attack attempts?
A：基于签名的入侵检测
Signature-based intrusion detection
B：基于异常的入侵检测
Anomaly-based intrusion detection
C：强大的补丁管理
Strong patch management
D：全盘加密
Full-disk encryption
答案:题目109的正确答案是：

**B：基于异常的入侵检测 (Anomaly-based intrusion detection)**

**解析：**
- **A：基于签名的入侵检测 (Signature-based intrusion detection)**：这种方法依赖于已知攻击的特征库来检测攻击。对于零日攻击（即尚未被发现或没有已知签名的攻击），基于签名的检测方法无法有效识别。
  
- **B：基于异常的入侵检测 (Anomaly-based intrusion detection)**：这种方法通过分析系统的正常行为模式，检测出与正常行为不符的异常活动。由于零日攻击通常表现出与正常行为不同的特征，基于异常的检测方法更有可能发现这些未知的攻击。

- **C：强大的补丁管理 (Strong patch management)**：虽然补丁管理对于防止已知漏洞被利用非常重要，但它无法直接检测零日攻击，因为零日攻击利用的是尚未被发现或修补的漏洞。

- **D：全盘加密 (Full-disk encryption)**：全盘加密主要用于保护数据在存储时的安全性，防止未经授权的访问。它与检测零日攻击无关。

因此，**基于异常的入侵检测**是最有效的控制措施，用于检测零日攻击。

A directive control; a one-way access corridor
